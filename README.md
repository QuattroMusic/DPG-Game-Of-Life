# DPG-Game-Of-Life

![](src/GMF.png)

The Conway's Game Of Life made in Python using DearPyGui

---
### Features

- A 70 x 40 grid
- Start, Pause And Stop Button
- Next Frame
- Wrapping
- Simulation Speed (from 0.05s to 2s)
