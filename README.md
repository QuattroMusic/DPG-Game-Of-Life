# DPG Conway's Game Of Life

![](src/GMF.ico)

The Conway's Game Of Life built in Python using DearPyGui

---
### Features

- A 70 x 40 grid
- Start, Pause And Stop Button
- Next Frame
- Wrapping
- Simulation Speed (from 0.02s to 2s)
- Save and Load the board
