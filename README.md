# DPG Conway's Game Of Life

![](src/GMF.ico)

---
### Features

- A 70 x 35 grid
- Start, Pause And Stop Button
- Next Frame
- Wrapping
- Changeable Refresh Rate, aka Simulation Speed (x0.5 to x100, where x1 = 1sec)
- Save and Load the board
